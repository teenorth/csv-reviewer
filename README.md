## Python virtual environment

Create virtual environment:

```bash
python3 -m venv venv
```

Activate virtual environment:

```bash
. venv/bin/activate
```

Freeze the latest package changes to the requirements file:

```bash
pip freeze > requirements.txt
```

Install packages:

```bash
pip install -r requirements.txt
```
